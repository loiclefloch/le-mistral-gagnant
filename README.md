# le-mistral-gagnant

This repository provides guidelines and tools for API development, including documentation and a custom linter to ensure best practices are followed. It is designed to help teams maintain high standards in API design and implementation.

## Features
- API guidelines documentation
- Linter for validating API guideline compliance

## Contributors
- LELO
- DEIS
- OLIV

## Structure
- `api_guidelines.md` / `api_guidelines_v2.md`: API guidelines documentation
- `linter/`: Contains the API guidelines linter script

## Usage
Refer to the documentation files for API standards. Use the linter script in the `linter/` directory to check your API code for compliance.
